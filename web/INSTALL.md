# Installation

Installation is currently handled via the wikijump-infra repo, documentation to
follow.

## Installing dependencies

### PHP

Install composer:

```shell
apt install composer
```

Install dependencies:

```shell
composer install
```

### JS

Install node and npm:

```shell
apt install nodejs
```

Install dependencies:

```shell
npm install
```

Transpile TypeScript to JavaScript:

```shell
npm run build
```
